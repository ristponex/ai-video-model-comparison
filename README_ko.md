## 1. 비교 기준 개요

이 벤치마크는 AI 비디오 생성 모델을 **3가지 핵심 축**으로 평가합니다:

### 축 1: 스펙 (기술 사양)

객관적으로 측정 가능한 기술 파라미터 — 해상도, 프레임 레이트, 길이, 모달리티 지원, 고유 기능. 모델이 *무엇을 할 수 있는지*를 결정합니다.

### 축 2: 가격 (비용 효율)

요청당 비용, 초당 비용, 실제 프로덕션 시나리오별 총비용. 공식 가격과 Atlas Cloud 가격을 비교하고, 품질 단위당 비용을 산출하여 최고의 가성비를 찾습니다.

### 축 3: 품질 (영상 및 오디오 충실도)

주관적이지만 체계적인 출력 품질 평가. 영상 품질, 프롬프트 충실도, 물리적 리얼리즘, 시간적 일관성, 오디오 동기화를 포함합니다. 표준화된 테스트 프롬프트를 기반으로 5점 척도로 평가합니다.

```
                    ┌─────────────────────┐
                    │       스펙           │
                    │  해상도, FPS,        │
                    │  길이, 모달리티      │
                    └────────┬────────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
    ┌─────────▼────┐  ┌─────▼──────┐  ┌───▼──────────┐
    │    가격      │  │   품질     │  │  사용 사례   │
    │  $/초        │  │  영상     │  │  시나리오    │
    │  $/초        │  │  오디오   │  │  매칭       │
    │  $/100개     │  │  물리     │  │              │
    └──────────────┘  └────────────┘  └──────────────┘
```

> **평가 방법**: 모든 품질 점수는 50개의 다양한 표준화 테스트 프롬프트(인물, 동물, 풍경, 추상 개념, 다중 요소, 물리 시나리오)를 기반으로 합니다. 각 모델은 프롬프트당 3회 테스트했으며, 점수는 평균값입니다.

## 7. 빠른 시작 — 모든 모델 체험

모든 모델이 동일한 API 포맷 사용. 모델 전환은 파라미터 하나.

### cURL

```bash
# Atlas Cloud API로 비디오 생성 — model 파라미터만 변경하면 모델 전환
curl -X POST "https://api.atlascloud.ai/v1/video/generate" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-v1.5-pro-t2v",
    "prompt": "한복을 입은 여성이 벚꽃길을 걷는다, 꽃잎이 바람에 흩날리며, 시네마틱 라이팅, 슬로우 모션",
    "duration": 5,
    "aspect_ratio": "16:9"
  }'
```

### Python

```python
import requests

# Atlas Cloud 통합 API — 하나의 키로 모든 모델 접근
API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.atlascloud.ai/v1/video/generate"

# 사용 가능한 모델 목록
MODELS = {
    "seedance": "seedance-v1.5-pro-t2v",      # $0.222/초부터 — 최고 오디오 동기화
    "kling_pro": "kling-v3.0-pro-t2v",         # $0.204/초부터 — 4K 화질
    "wan": "wan-2.6-t2v",                       # $0.07/초부터 — 가성비 최고
    "veo": "veo-3.1-t2v",                       # $0.18/초부터 — Google 품질
    "hailuo": "hailuo-2.3-pro-t2v",            # $0.49/초부터 — 빠른 생성
}

def generate_video(model_key: str, prompt: str, duration: int = 5):
    """
    AI 비디오 생성 — Atlas Cloud 통합 API 사용

    인자:
        model_key: 모델 식별자 (MODELS 딕셔너리 참조)
        prompt: 비디오 설명 프롬프트
        duration: 비디오 길이 (초)
    """
    response = requests.post(
        BASE_URL,
        headers={
            "Authorization": f"Bearer {API_KEY}",
            "Content-Type": "application/json",
        },
        json={
            "model": MODELS[model_key],
            "prompt": prompt,
            "duration": duration,
            "aspect_ratio": "16:9",
        },
    )
    return response.json()


# 예시: 다른 모델로 같은 프롬프트의 비디오를 생성하여 비교
prompt = "모던 키친에서 셰프가 디저트를 플레이팅한다. 초콜릿 소스에서 증기가 피어오른다. 얕은 피사계 심도, 따뜻한 조명, 4K 시네마틱."

# Wan 2.6으로 생성 (최저가)
result_wan = generate_video("wan", prompt)
print(f"Wan 2.6 결과: {result_wan}")

# Kling 3.0 Pro로 생성 (최고 화질)
result_kling = generate_video("kling_pro", prompt)
print(f"Kling 3.0 Pro 결과: {result_kling}")
```
