## 1. 比較の軸

本ベンチマークでは、AI動画生成モデルを**3つの主要軸**で評価します：

### 軸1：スペック（技術仕様）

客観的に測定可能な技術パラメータ — 解像度、フレームレート、尺、モダリティ対応、独自機能。モデルが*何をできるか*を決定します。

### 軸2：料金（コスト効率）

リクエスト単価、秒単価、実運用シナリオでの総コスト。公式料金、Atlas Cloud料金を比較し、品質単位あたりのコストを算出して最良のバリューを見つけます。

### 軸3：品質（映像・音声忠実度）

主観的ながら体系化された出力品質の評価。映像品質、プロンプト忠実度、物理リアリズム、時間的コヒーレンス、音声同期をカバー。標準化されたテストプロンプトに基づき5段階で評価します。

```
                    ┌─────────────────────┐
                    │     スペック         │
                    │  解像度、FPS、       │
                    │  尺、モダリティ      │
                    └────────┬────────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
    ┌─────────▼────┐  ┌─────▼──────┐  ┌───▼──────────┐
    │    料金      │  │   品質     │  │ ユースケース │
    │  $/リクエスト │  │  映像     │  │  シーン      │
    │  $/秒        │  │  音声     │  │  マッチング  │
    │  $/100本     │  │  物理     │  │              │
    └──────────────┘  └────────────┘  └──────────────┘
```

> **評価方法**：全品質スコアは50の多様なテストプロンプト（人物、動物、風景、抽象概念、マルチエレメント、物理シーン）に基づいています。各モデルはプロンプトごとに3回テストし、スコアは平均値です。

### Python

```python
import requests

# Atlas Cloud統合API — 1つのキーで全モデルにアクセス
API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.atlascloud.ai/v1/video/generate"

# 利用可能なモデル一覧
MODELS = {
    "seedance": "seedance-v1.5-pro-t2v",      # $0.222/リクエスト — 最高の音声同期
    "kling_pro": "kling-v3.0-pro-t2v",         # $0.204/リクエスト — 4K品質
    "wan": "wan-2.6-t2v",                       # $0.07/リクエスト — コスパ最高
    "veo": "veo-3.1-t2v",                       # $0.18/リクエスト — Google品質
    "hailuo": "hailuo-2.3-pro-t2v",            # $0.49/リクエスト — 高速生成
}

def generate_video(model_key: str, prompt: str, duration: int = 5):
    """
    AI動画を生成 — Atlas Cloud統合APIを使用

    引数:
        model_key: モデル識別子（MODELSディクショナリ参照）
        prompt: 動画の説明プロンプト
        duration: 動画の長さ（秒）
    """
    response = requests.post(
        BASE_URL,
        headers={
            "Authorization": f"Bearer {API_KEY}",
            "Content-Type": "application/json",
        },
        json={
            "model": MODELS[model_key],
            "prompt": prompt,
            "duration": duration,
            "aspect_ratio": "16:9",
        },
    )
    return response.json()


# 例：異なるモデルで同じプロンプトの動画を生成して比較
prompt = "モダンキッチンでシェフがデザートを盛り付ける。チョコレートソースから蒸気が立ち上る。浅い被写界深度、暖かい照明、4Kシネマティック。"

# Wan 2.6で生成（最安）
result_wan = generate_video("wan", prompt)
print(f"Wan 2.6 結果: {result_wan}")

# Kling 3.0 Proで生成（最高画質）
result_kling = generate_video("kling_pro", prompt)
print(f"Kling 3.0 Pro 結果: {result_kling}")
```

## 8. よくある質問

### 2026年最高のAI動画生成AIは？

優先事項次第：
- **最高品質**：**Kling 3.0 Pro** — ネイティブ4K、60fps
- **コスパ最高**：**Wan 2.6** — $0.07/リクエスト
- **最高音声同期**：**Seedance v1.5 Pro** — 8言語リップシンク
- **エンタープライズ向け**：**Veo 3.1** — Google安全基準

### SeedanceとKling、どちらが良い？

**Seedance v1.5** は音声同期、カメラ制御、多言語リップシンクで優位。**Kling 3.0** は映像品質（4K、60fps）、物理リアリズム、マルチショットで優位。音声コンテンツにはSeedance、映像コンテンツにはKling。

### 最安のAI動画生成APIは？

[Atlas Cloud](https://www.atlascloud.ai?ref=JPM683) 経由：
1. **Wan 2.2 Spicy**: $0.03/リクエスト（NSFW、I2Vのみ）
2. **Wan 2.6**: $0.07/リクエスト（汎用、最高コスパ）
3. **Veo 3.1**: $0.18/リクエスト（Google公式比90%オフ）

### NSFW AI動画は生成できる？

はい — **Wan 2.2 Spicy I2V** が [Atlas Cloud](https://www.atlascloud.ai?ref=JPM683) で$0.03/リクエストで利用可能。画像→動画のみ対応。

### 無料トライアルはある？

Atlas Cloudは**初回チャージ25%ボーナス**（最大$100）を提供。$100チャージで$125のクレジット — 約1,785本のWan 2.6動画または約562本のSeedance v1.5動画に相当。

### どのモデルがSNSコンテンツに最適？

**Wan 2.6** が大量のTikTok/Reels/Shorts向けコンテンツに最適 — 全SNSアスペクト比（9:16、1:1、16:9）対応で1本$0.07。プレミアムSNSコンテンツには **Seedance v1.5** の音声同期が制作品質を大幅に向上。

### Kling vs Sora：どちらを選ぶべき？

2026年3月時点で、**Kling 3.0 Pro** はSoraを解像度（4K vs 1080p）、フレームレート（60fps vs 24fps）で上回り、マルチショット構成も提供。KlingはAtlas Cloudのような第三者APIを通じて85%割引でアクセス可能。

### モデル間の切り替え方法は？

Atlas Cloudは統合APIを使用。リクエスト内の `model` パラメータを変更するだけ：
```bash
# モデル切替はパラメータ1つだけ
"model": "wan-2.6-t2v"            # → Wan 2.6
"model": "kling-v3.0-pro-t2v"     # → Kling 3.0 Pro
"model": "seedance-v1.5-pro-t2v"  # → Seedance v1.5
```
