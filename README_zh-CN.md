## 1. 对比维度概述

本评测报告从**三个核心维度**对 AI 视频生成模型进行全面评估：

### 维度一：规格（技术能力）

客观可量化的技术参数 — 分辨率、帧率、时长、模态支持和独特功能。这些决定了模型*能做什么*。

### 维度二：价格（成本效益）

单次请求成本、每秒成本以及实际生产场景下的总成本。对比官方定价、Atlas Cloud定价，并计算单位质量成本以找到最佳性价比。

### 维度三：质量（视觉与音频保真度）

主观但系统化的输出质量评估，涵盖视觉清晰度、提示词遵循度、物理真实感、时间连贯性和音频同步。每个维度基于标准化测试提示词以5星制评分。

```
                    ┌─────────────────────┐
                    │     规格参数         │
                    │  分辨率、帧率、      │
                    │  时长、模态          │
                    └────────┬────────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
    ┌─────────▼────┐  ┌─────▼──────┐  ┌───▼──────────┐
    │    价格      │  │   质量     │  │  使用场景    │
    │  $/秒        │  │  视觉     │  │  场景匹配    │
    │  $/秒        │  │  音频     │  │              │
    │  $/100个视频 │  │  物理     │  │              │
    └──────────────┘  └────────────┘  └──────────────┘
```

> **评测方法**：所有质量评分基于50个标准化测试提示词，涵盖人物、动物、风景、抽象概念、多元素场景和物理密集场景。每个模型对每个提示词测试3次，评分取平均值。

```python
import requests

# Atlas Cloud统一API — 一个密钥访问所有模型
API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.atlascloud.ai/v1/video/generate"

# 所有可用模型 — 切换只需更改model字段
MODELS = {
    "seedance": "seedance-v1.5-pro-t2v",      # $0.044/秒起 — 最佳音频同步
    "kling_pro": "kling-v3.0-pro-t2v",         # $0.204/请求 — 4K画质

    "wan_spicy": "wan-2.2-spicy-i2v",          # $0.03/请求 — 无审查
    "veo": "veo-3.1-t2v",                       # $0.18/请求 — Google品质
    "hailuo": "hailuo-2.3-pro-t2v",            # $0.49/请求 — 快速生成
    "vidu": "vidu-q3-pro-t2v",                  # 参考图一致性
}

def generate_video(model_key: str, prompt: str, duration: int = 5):
    """
    生成AI视频 — 通过Atlas Cloud统一API

    参数:
        model_key: 模型标识符（见MODELS字典）
        prompt: 视频描述提示词
        duration: 视频时长（秒）
    """
    response = requests.post(
        BASE_URL,
        headers={
            "Authorization": f"Bearer {API_KEY}",

            "prompt": prompt,
            "duration": duration,
            "aspect_ratio": "16:9",
        },
    )
    return response.json()
```
